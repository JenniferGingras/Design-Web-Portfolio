Solving a snowboarder's dilemma.



Stay in the Snow

There is no single place to find current information on multiple ski mountains. This is a team created application that makes it easy to decide on a mountain to go to for a day at the slopes. It allows users to search multiple ski mountains and receive and compare data on weather conditions, list status, and distance from the user's location.

Technologies:
HTML, CSS, JavaScript, APIs, Firebase
